# Power One Mall - Local Backups Directory

This directory stores local backup snapshots generated by the automated backup scripts.

## Directory Structure

- `daily/`: Daily automated backups (retains historical folders)
- `weekly/`: Weekly automated backups (retains historical folders)
- `archives/`: Compact `.zip` files containing full project archives

## Safety Information

This directory is **excluded from Git version control** (via `.gitignore`) to prevent repository bloat and secret leakage. Do not remove the `.gitignore` file from this folder.
